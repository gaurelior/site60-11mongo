MANUAL DO SITE

Este site foi desenvolvido para ser informativo, de fácil navegação e com uma interface amigável. A página inicial apresenta um layout clean e intuitivo, com botões interativos que permitem o acesso às demais seções do site.

O site é composto por duas páginas adicionais. Ao clicar no botão à direita, você será direcionado para a página "Seja um Parceiro", que foi projetada para facilitar a inscrição. Essa página possui um fundo interativo e oferece dois botões principais: um para acessar o "Contrato de parceria" e outro para preencher o "Formulário de inscrição". Caso o visitante deseje se tornar parceiro de forma rápida e prática, ele pode preencher o formulário, que está integrado a um banco de dados administrado pelos membros do projeto.

Além da página de inscrição, o site também conta com uma página de informações detalhadas sobre o projeto, incluindo dados sobre os parceiros, e quem são os autores do projeto entre outras informações relevantes.
